# Week 1 Findings

## Key Findings
- Month-to-Month contracts have highest churn.
- Low tenure customers churn more.
- Higher monthly charges increase churn risk.
- Customers without Tech Support churn more.
- Customers without Online Security churn more.
- Electronic Check users show higher churn.
- Fiber Optic customers have higher churn.

## Business Recommendations
- Target Month-to-Month customers.
- Improve onboarding for new customers.
- Offer loyalty discounts.
- Promote auto-pay.
- Bundle Tech Support and Security services.
